**Q4. Which employee does the IP address 10.10.0.216 belong to?**<br><br>

```
Employees
| where ip_addr == "10.10.0.216"
```
<br>

**Answer: Nakia Acosta**<br><br>

**Q5. How many emails did Nakia Acosta receive?**<br><br>

```
Email
| where recipient == "nakia_acosta@encryptoderafinancial.com"
| count
```
<br>

**Answer: 38**<br><br>

**Q6. How many distinct senders were seen in the email logs from bitbingersbanking.net?**<br><br>

```
Email
| where sender has "bitbingersbanking.net"
| distinct sender
| count
```
<br>

**Answer: 1204**<br><br>

**Q7. How many distinct websites did Timothy Geffre visit?**<br><br>

```
let timothyIP =
Employees
| where name contains "Timothy Geffre"
| distinct ip_addr;
OutboundNetworkEvents
| where src_ip in (timothyIP)
| distinct url
| count 
```
<br>

**Answer: 92**<br><br>

```

| where domain contains "money"
| distinct domain
| count
```

**Answer: 15**<br><br>
